# llama-starter

1. 克隆项目
1. 下载 llama.cpp 程序，然后放入项目根目录 [github releases 页面](https://github.com/ggml-org/llama.cpp/releases)
1. 运行 `./llama-starter.bat`

## models

1. 从可信仓库下载 `gguf` 模型，例如： [Hugging Face](https://huggingface.co/)
2. 示例模型下载： [Qwen3.6-35B-A3B-Uncensored-HauhauCS-Aggressive](https://huggingface.co/HauhauCS/Qwen3.6-35B-A3B-Uncensored-HauhauCS-Aggressive)
3. 将下载的 `.gguf` 文件放入 `models/` 目录
